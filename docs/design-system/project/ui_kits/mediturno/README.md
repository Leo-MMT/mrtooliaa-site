# MediTurno IA UI kit

WhatsApp appointment-booking bot for clinics. Color system: **medical blue** + **WhatsApp green**.

- `Components.jsx` — `Navbar`, `Hero`, `WaIcon`, and the interactive `ChatMockup` (click "Siguiente mensaje" to advance through the booking flow).
- `Sections.jsx` — `Benefits` (4-up feature grid), `Pricing` (2-tier Básico/Pro), `Footer`.
- `kit.css` — shared tokens, gradient-med, gradient-wa, pulsing WhatsApp CTA.
