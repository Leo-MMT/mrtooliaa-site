# MR ToolIAA UI kit

Corporate landing recreation. Components:
- `Navbar.jsx` — transparent-over-hero nav that turns navy+blurred on scroll.
- `Hero.jsx` — split hero with gradient bg, glow orbs, "Arsenal" glass card and floating notification.
- `ProblemSolution.jsx` — three-up pain / bridge / result cards.
- `ProductCards.jsx` — the three product offerings (MediTurno, Ferret-Quote, Custom). The first card is the "active" one with orange border.
- `Footer.jsx` — deep-navy 3-col footer.

Open `index.html` to see them composed.
