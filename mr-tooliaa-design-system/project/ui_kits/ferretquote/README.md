# Ferret-Quote UI kit

Industrial orange + slate for hardware-store WhatsApp bot. Bolder and darker than MediTurno.

- `Components.jsx` — `Navbar`, `Hero` with side-by-side WhatsApp mockup + floating PDF card (rotated).
- `Sections.jsx` — `HowItWorks` (4-step grid with number badges), `Pricing` (dark Básico / light Pro), `Footer` with floating WhatsApp button.
- `kit.css` — `.hero-bg` diagonal texture, `.stripe-accent` construction warning stripe, btn-orange / btn-dark / btn-outline.
